# MoAI  
**자료 공유 → AI 요약·정리 → 화상 발표까지 한 번에 지원하는 온라인 스터디 플랫폼**  

이 플랫폼은 **스터디 자료를 한 곳에 모아 관리하고, AI 요약·구조화 기능과 실시간 화상 발표(WebRTC)를 통합 제공**하여  
효율적이고 몰입도 높은 학습 환경을 지원합니다.

---

##  주요 기능  

### 1. 스터디 공부 자료 공유 + 정리  
- 스터디원이 업로드한 **공부(원본) 자료 저장·열람·다운로드 가능**  
- **사용자 / 날짜 / 카테고리 필터링**을 통해 원하는 자료만 선택 후 AI 요약 자료 생성  
- **AI가 자료를 요약, 핵심 키워드 추출, 구조화**  
- **공부(원본) 자료와 AI 요약 자료 병렬 제공**  
- AI 요약 자료의 특정 내용을 클릭하면 **공부(원본) 자료의 해당 페이지로 이동**  
- 공부(원본) 자료 + AI 요약 자료(공유)를 **한 곳에서 통합 관리**  

**AI 요약 파이프라인**  
- 사용자가 공부(원본) 자료 업로드  
- Gemini 기반 AI가 자료를 **템플릿 형태로 요약 및 구조화** 
- 사용자가 **AI 모델/요약본 생성 방식** 옵션으로 선택 및 기입 가능 
- AI 요약 자료과 공부(원본) 자료 매핑 → **클릭 시 공부(원본) 자료 위치로 이동 가능**  

---

### 2. 화상 스터디(WebRTC)  
- 스터디원들과 **실시간 화상 참여**  
- 공부(원본) 자료 + AI 요약 자료(공유)를 **바로 불러와 발표·토론 가능**  

---

### 3. 통합 학습 환경  
- **스터디 공부(원본) 자료 필터링 후 선택 요약**  
  → 전체 자료 중 특정 사용자/날짜/카테고리 기반으로 원하는 자료만 선택  
- **통합된 자료 관리 공간**  
  → 스터디 공부(원본) 자료 + AI 요약 자료(공유)를 한 곳에서 조회·정리
  → AI 요약 자료(개인) 조회·정리  
- **화상 발표와 연계된 학습 환경**  
  → 스터디 공부(원본) 자료 + AI 요약 자료(공유)를 실시간 화상회의에서 바로 공유·발표 가능
